This is a personal portfolio website for Rene Naidoo, a final-year BSc IT student specializing in web development. The website showcases Rene’s skills, projects, experience, and provides a way to get in touch via a contact form.

It’s built using HTML, CSS, and JavaScript, with a focus on responsive design, light/dark mode, and user-friendly navigation.

Features

Home Page

Introduction with a greeting and short bio.

Profile image and “View My Projects” button.

Clean, responsive layout with a background image.

Light/Dark mode toggle stored in local storage.

About Page

Detailed “About Me” section.

Downloadable CV link.

Education, skills, experience, and interests sections.

Skills displayed in interactive cards with hover effects.

Projects Page

Showcases key projects with descriptions.

Each project includes tech stack, “View Code” link, and “Live Demo” link.

Project cards with hover effects for better user experience.

Contact Page

Contact form to submit name, email, and message.

Profile image included.

LinkedIn link for professional networking.

Light/Dark mode toggle consistent across all pages.

Global Features

Responsive design for desktops, tablets, and mobile screens.

Flexbox and Grid layouts for structured sections.

Smooth transitions for hover effects and theme toggling.

Footer consistent across all pages.

Tech Stack

HTML5 – Semantic structure for web pages.

CSS3 – Styling with responsive design, Grid, and Flexbox.

JavaScript – Light/Dark mode toggle with local storage persistence.
